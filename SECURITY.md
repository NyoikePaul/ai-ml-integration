# Security Policy

## Reporting Vulnerabilities

Report to: security@nyoikepaul.dev

Include: Description, location, severity, proof of concept, impact

Response time: 24 hours acknowledgment

## Best Practices
- Keep dependencies updated
- Use environment variables for secrets
- Validate all user input
- Enable authentication on endpoints
- Rotate API keys regularly

## Compliance
- OWASP Top 10
- CWE/SANS Top 25
- NIST cybersecurity
- GDPR where applicable

## Security Scanning
- CodeQL analysis
- Bandit scanning
- Dependabot updates
- SAST & DAST testing

All code scanned on every push.
